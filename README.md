## Extension Settings

This extension contributes the following settings:

-   `lupex.timer`: enable/disable the timer feature
-   `lupex.countDown.enabled`: A countdown that sits in the status bar and tells you when is time to get up and do some stretching.
-   `lupex.countDown.redTextLastTotSeconds`: The last tot seconds of the count down are displayed red.
